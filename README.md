# 오엘베이비 (Olbaby) - 아기사진 스튜디오 웹사이트 리디자인

CRO(전환율 최적화) 원칙을 적용하여 재설계된 프리미엄 베이비 스튜디오 웹사이트입니다.

## 주요 구조 (10개 섹션)
1. **히어로 (Hero)**: 사회적 증거 및 듀얼 CTA
2. **문제 공감 (Empathy)**: 고객 상황 체크리스트
3. **가치 제안 (USP)**: 3가지 차별점 (안전, 감성, 올인원 패키지)
4. **포트폴리오 (Portfolio)**: 만삭, 백일, 돌 등 주요 테마 화보
5. **촬영 과정 (Process)**: 예약부터 수령까지 4단계
6. **촬영 안내 (Pricing)**: 앵커링 효과를 적용한 가격 비교표 
7. **후기 (Reviews)**: 만족도 통계 및 실제 고객 리뷰 카드
8. **자주 묻는 질문 (FAQ)**: 아코디언 UI를 활용한 고객 불안 해소
9. **한정 오퍼 (Offer)**: 월말 카운트다운 및 혜택
10. **연락처 (Contact)**: 연락처, 카카오톡, 오시는 길

## 기술 스택
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
